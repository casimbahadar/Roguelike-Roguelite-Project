# Roguelike-Roguelite-Project

Four Fire-Emblem-foundation tactics-roguelites sharing one
Godot 4 core. Same engine, four theme packs, sequential ship
order.

- **Game 1 — Sengoku** (working title *Banner of Ashes*): grid
  tactics with permadeath and a Slay-the-Spire-style branching
  campaign, set in an alternate Sengoku Japan with original
  characters and public-domain historical warlords.
- **Game 2 — FE-foundation high fantasy** (working title
  *Crystalfall Tactics*): same engine, Fire-Emblem mechanics
  chassis, Final-Fantasy / classic-JRPG flavor (job system,
  crystals, summon archetypes) under fully original names,
  designs, and lore.
- **Game 3 — FE-foundation creature collection** (working title
  *Pocketkin Tactics* — *Pocket* + *kin* telegraphs the
  Pokemon-genre creature-collection theme): same engine, FE
  chassis + a creature-collection loop (recruitable original
  beasts called *pocketkin*, element triangle overlaying the
  weapon triangle, multi-stage promotions, trainer–beast bonds
  as relics). No Pokemon IP — see `CLAUDE.md` for the locked
  guardrails.
- **Game 4 — FE-foundation partner-bond** (working title
  *Datapact Tactics* — *Data* + *pact* telegraphs the
  Digimon-genre partner-bond theme): same engine, FE chassis +
  a partner-bond loop (one evolving *vessel* paired with each
  lord, four-stage promotion gated by story flags). No Digimon
  IP — see `CLAUDE.md` for the locked guardrails.

Shared `core/` plus theme packs under `games/sengoku/`,
`games/crystal/`, `games/pocketkin/`, and `games/datapact/`.
All four working titles are placeholders pending a brainstorm
and formal selection later; Games 3 and 4 specifically need
heightened TESS / EUIPO clearance because their genres are
dominated by entrenched franchises.

**Sequencing** (locked in `CLAUDE.md`): Game 1 ships first.
Game 2 begins after Game 1's slice is internally validated.
Games 3 and 4 enter pre-production only after Game 1 or Game 2
has shipped — they reuse a hardened combat / run / meta layer.
No parallel full-team work on more than one game at a time.

Active development happens on
`claude/fire-emblem-roguelike-games-FadWA`. See `CLAUDE.md` for
the locked project rules.
