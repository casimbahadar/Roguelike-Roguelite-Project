# Roguelike-Roguelite-Project

Two Fire-Emblem-inspired tactics-roguelites sharing one Godot 4 core:

- **Game 1 — Sengoku** (working title *Banner of Ashes*): grid tactics
  with permadeath and a Slay-the-Spire-style branching campaign,
  set in an alternate Sengoku Japan with original characters and
  public-domain historical warlords.
- **Game 2 — FE-foundation high fantasy** (working title
  *Crystalfall Tactics*): same engine, Fire-Emblem mechanics chassis,
  Final-Fantasy / classic-JRPG flavor (job system, crystals, summon
  archetypes) under fully original names, designs, and lore.

Shared `core/` plus theme packs under `games/sengoku/` and
`games/crystal/`. Working titles are placeholders pending
trademark search.

Active development happens on
`claude/fire-emblem-roguelike-games-FadWA`. See `CLAUDE.md` for the
locked project rules.
